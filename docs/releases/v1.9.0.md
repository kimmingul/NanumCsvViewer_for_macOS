# Nanum CSV Viewer v1.9.0

## Highlights

A feature release adding export, appearance, and benchmarking controls.

- **Export options** — the export save panel offers an encoding choice (UTF-8, UTF-8 with BOM, CP949 / EUC-KR) and a "Reveal in Finder after export" checkbox, and shows progress. Encoding and BOM apply to CSV only; JSON/HTML/Markdown stay UTF-8. A CP949 export that must substitute an unrepresentable character warns instead of dropping it silently.
- **Appearance** — View ▸ Appearance forces System / Light / Dark app-wide, remembered across launches.
- **Font size** — View ▸ Font Size (Small / Medium / Large) scales grid text and the row-number gutter independently of row density, with the row height growing so text never clips.
- **Benchmark** — View ▸ Run Benchmark (⌥⌘B) times read-only full-scan, search, and distinct passes over the current view and reports time / rows / rows·s⁻¹ plus a total; repeatable, mutates nothing.
- Reviewed by Codex and Grok; findings adjudicated and fixed.
- Bumped bundle metadata to `1.9.0(190)`.
