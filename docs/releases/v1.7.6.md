# Nanum CSV Viewer v1.7.6

## Highlights

- Fixed Pivot Builder chart hover tooltips so they stay compact and readable over bar and line charts.
- Positioned hover tooltips from the selected chart mark instead of a fixed top-right overlay.
- Changed tooltip rendering to use an opaque app background and disabled hit testing so mouse tracking remains stable.
