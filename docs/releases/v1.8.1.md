# Nanum CSV Viewer v1.8.1

## Highlights

- Multiple named saved views (bookmarks) per file: `Save View As...`, a `Restore Saved View...` picker with delete, and a `Restore View on Open` toggle that reapplies the most recent bookmark. Existing single saved views migrate automatically.
- Column management: a `View > Columns` visibility checklist, per-file drag-reorder persistence, and exports that follow the on-screen column order.
- `View > Row Density` (Compact / Regular / Comfortable) row heights, and a process memory-footprint line in the performance dashboard.
- Pin Column to Front: a header right-click keeps an identifier column just after the row-number gutter, persisted per file with a pin marker — the lightweight alternative to full Excel freeze panes.
- Hardened per Codex and Grok adversarial reviews: saved-view recency on delete, keep-one-column-visible on every path, the row-number gutter pinned against reorder, strict Columns-menu window binding, corrupt-store no-legacy-regression, per-file map pruning, and skipping the busy scan for filter-free auto-restore.
- Bumped bundle metadata to `1.8.1(181)`.
