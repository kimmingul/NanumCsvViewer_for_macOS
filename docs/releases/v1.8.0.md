# Nanum CSV Viewer v1.8.0

## Highlights

- Reached feature parity with the Windows twin v1.15.
- Added the Facets panel (View ▸ Facets Panel, F6): 6-bin histograms for numeric columns and top-6 value bars for other columns, with bar-click cross-filtering (AND across columns, OR within a column) that keeps the active selection visible and toggleable.
- Added the Visualization menu with seven modeless Swift Charts windows: histogram + KDE + Shapiro-Wilk badge, grouped boxplot + ANOVA, scatter + OLS fit (density grid past 20k points), correlation heatmap, normal Q-Q plot, date-binned time series, and Pareto with cumulative percent.
- Added the Data Quality module (Cmd+Shift+P): a full-file profiler with sentinel, type-validity, key-uniqueness, ragged-row, and duplicate-row rules, a categorical codebook, a 0-100 score, and Markdown/HTML/JSON report export.
- Added Excel .xlsx/.xlsm import with a multi-sheet picker, built on a dependency-free ZIP/XML reader with shared strings and 1900/1904 date serial support.
- The Inspector is visible by default on first launch (toggle with F4); analysis, chart, and pivot scans cap at 2,000,000 rows with a visible scope note.
- Hardened per Codex and Grok adversarial reviews: Excel header rows pad to the sheet's true width, ZIP entry guards, KDE/Shapiro-Wilk sampling caps, collision-safe duplicate hashing, and the logout-shortcut collision fix.
- Bumped bundle metadata to `1.8.0(180)`.
