Everything you need to read the file before it reads your day.

Built for large CSV workflows, with a virtual table, background indexing, native search, pivot tools, and charts.

Large-file first.

Byte-offset indexing and virtual row rendering keep multi-gigabyte CSV files responsive without loading everything into memory.

Search that adapts.

Use plain text, regular expressions, slash regex, or fuzzy search, then combine results with column and selected-cell filters.

Pivot in place.

Create pivot tables and native Swift Charts output with multiple measures, date grouping, filters, totals, chart modes, and wide result-pane charts.

Reads real CSV.

Handles UTF-8, UTF-8 BOM, CP949/EUC-KR, quoted newlines, escaped quotes, and malformed first-row quotes.

Export the view.

Share the current filtered and sorted view as CSV, Markdown, JSON, or HTML with visible columns only.

Designed for macOS.

Native menus, tabs, drag and drop, clipboard import, light and dark appearance, and a notarized Developer ID release.

Your CSV stays on your Mac.

Nanum CSV Viewer is a local desktop app. It opens files you choose, builds local indexes, and does not need a cloud service to inspect or analyze your data.
