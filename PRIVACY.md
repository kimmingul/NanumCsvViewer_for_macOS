# Nanum CSV Viewer Privacy Policy

Effective date: June 27, 2026

Nanum CSV Viewer is a local macOS app for opening, viewing, filtering, analyzing, and exporting CSV and text files.

## Data Collection

Nanum CSV Viewer does not collect personal data.

The app does not use analytics SDKs, advertising SDKs, tracking technologies, user accounts, telemetry services, crash-reporting services, or remote servers.

## Local File Access

CSV files are opened from locations selected by the user. File contents are processed locally on the Mac.

The app may create local index cache files to make large CSV files faster to reopen. These files stay on the user's Mac and can be deleted from the app's settings.

## Clipboard and Exports

The app can read CSV text or file paths from the clipboard when the user chooses to open from clipboard. The app can also copy selected values or analysis results to the clipboard when the user chooses a copy command.

Exports are saved only to locations selected by the user.
